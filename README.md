# The Price is Right

## Problem brief

1. Pick a number
1. Submit guesses
1. Reveal the number
1. Reveal the winning guess

## User(s) interface(s)

### Price Setter(P)

- Sets the price to be guessed.
- Sets duration of game.
- Set the wager.
- Reveals the winning guess.
- Who won?

### Guessers(G)

- Make a guess.
- If deadline not passed (game on).
- Submit the guess.
- Winners are funded.
- Losers lose wager.
